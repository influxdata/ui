For more detailed and up to date information check out the [GitHub Repository](https://github.com/influxdata/influxdb-client-java/tree/master/client-scala)

##### Add Dependency

Build with sbt

```
libraryDependencies += "com.influxdb" % "influxdb-client-scala" % "6.3.0"
```

Build with Maven

```
<dependency>
  <groupId>com.influxdb</groupId>
  <artifactId>influxdb-client-scala</artifactId>
  <version>6.3.0</version>
</dependency>
```

Build with Gradle

```
dependencies {
  implementation "com.influxdb:influxdb-client-scala:6.3.0"
}
```
